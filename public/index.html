Chat Support ============ Overview -------- This is a task for UA Web Challenge contest 2012, position: Frontend Engineer. http://uawebchallenge.com [Details TBD] Installation -------- * Clone the repo ``` got clone https://github.com/akhabibullina/chat_support.git ``` * Under console window go to the project directory and install node modules ``` npm install ``` * Run node ``` node ./api.js ``` or ... ``` node start ``` * The content will be available via ``` localhost:8080/client.html ``` Notes -------- - Made in jQuery plugin shape - Use socketIO as a dependency - Uses WebSockets
